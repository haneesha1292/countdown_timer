A simple countdown timer that has 4 pages. 
Page 1: Has a simple start button
page 2: Countdown timer with pause and exit button. When time is paused, the page gets redirected to resume page. When exit button is clicked, page is redirected to exit page
page 3: Resume page. Shows the remaining time.Has resume button which when clicked, redirects to previous page and timer starts again
page 4: Exit page. Contains a simple thank you message.
